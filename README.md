# Мобильное приложение для редактирования графических объектов интерфейса
**Дисциплина:** ПМ.04 Геймдизайн и разработка игр  
**Студент:** Ибадуллаева Дарина Мамыржановна  

##  Описание проекта
Данный проект разработан в рамках учебной практики. Приложение представляет собой мобильный редактор, позволяющий геймдизайнерам проектировать элементы игрового интерфейса (UI), настраивать их параметры и сохранять макеты для дальнейшего использования в игровых движках.

##  Реализованный функционал (CRUD)
В приложении реализован полный цикл управления графическими объектами:
* **Create (Создание):** Добавление кнопок, панелей и текстовых полей на рабочую область.
* **Read (Чтение):** Визуальное отображение списка всех созданных элементов.
* **Update (Изменение):** Редактирование координат, размеров и цветов объектов.
* **Delete (Удаление):** Очистка рабочей области или удаление конкретных элементов.

##  Работа с данными
Для обеспечения сохранности проектов реализована система **сериализации данных в JSON-файл**. Это позволяет:
1. Сохранять текущую конфигурацию интерфейса в файл `project_save.json`.
2. Загружать ранее созданные макеты при повторном запуске приложения.

##  Технический стек
* **Язык:** Python (использование библиотек для кроссплатформенной мобильной разработки).
* **Формат данных:** JSON.
* **Методология:** Компонентно-ориентированный подход к разработке UI.
